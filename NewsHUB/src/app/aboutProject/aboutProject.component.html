News HUB

The application contains news from a large number of news websites. These messages are obtained using News API. On the main page there is an overview of the most popular and the latest news.The application allows you to choose the offered by News API news category, search information and create an account that will enable personalization of displayed content.

Account data is stored by using a NoSQL database - Firebase. Logging in to the application is possible thanks to the traditional method (email, password) and through the account on the social networking site Facebook, or Google account.

The news on the registered user's home page is customized to his preferences. User will decide which categories will be displayed on the home screen. On the side screen are available information about the weather in the city, which the user chose on the profile page. This functionality will be possible thanks to the use Open Weather Map API.

The management of project work on the application has been happening on the platform Trello (link),furthermore the Git version control system was used to work with the files and also platform Github.

The application was implemented on the basis of the prototype, which can be viewed by clicking on this link, made in the program Axure RP. The password for previewing the prototype is a phrase druzynapierscienia.
